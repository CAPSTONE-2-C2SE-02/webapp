## Install

1. Clone the repository:
```sh
git clone https://github.com/CAPSTONE-2-C2SE-02/webapp.git
cd webapp
```
2. Install dependencies for both client and server:
```sh
npm run install:all
```
3. Start development servers:
```sh
npm run dev
```
